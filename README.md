# Plan Win Bot

Телеграм-бот для планирования с управлением через inline-кнопки в одном сообщении.

## Возможности
- Планы на день / неделю / месяц / год.
- Ежедневные дефолтные задачи: "Почитать Библию" и "Принять причастие".
- Молитвенный план по дням недели с автоподстановкой в дневной план.
- Добавление, редактирование, удаление задач.
- Замочки для закрепления задач (переносятся в новый период того же типа).
- Галочки выполнения по каждому периоду.
- Итоговая история по периодам (выполнено/не выполнено).
- Экспорт в Excel.
- Отдельные данные для каждого пользователя.
- Настраиваемые уведомления.
- Просмотр прогресса участников для администратора.

## Установка

1) Создайте бота через BotFather и получите токен.
2) Создайте `.env` рядом с `bot.py`:

```
BOT_TOKEN=ваш_токен
ADMIN_USER_ID=ваш_telegram_id
DB_PATH=data/bot.db
```

3) Установите зависимости:

```
pip install -r requirements.txt
```

4) Запустите бота:

```
python bot.py
```

## Использование
- `/start` — открыть главное меню.
- `/menu` — вернуться в меню.

Все управление — через inline-кнопки в одном сообщении.

### Правила планирования
- День: редактирование только сегодня.
- Неделя: редактирование только в понедельник текущей недели.
- Месяц: редактирование только 1–2 числа текущего месяца.
- Год: редактирование только в январе (после 2-го числа — с подтверждением).
- Закрепленные задачи автоматически попадают в следующий период.

## Примечания
- Уведомления работают по времени сервера (HH:MM).
- База данных хранится в `data/bot.db`.
- Экспорт отправляется файлом `.xlsx` и сохраняется в `exports/`.
