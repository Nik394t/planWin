# Daily

Мобильное приложение-планер для Android (с базой под iOS), воссозданное из логики `plan_win_bot`.

## Что реализовано

- Периоды: день, неделя, месяц, год.
- Задачи: добавление, редактирование, удаление, выполнение.
- Закрепление задач (замочки) и автоперенос через шаблоны.
- Молитвенный план по дням недели с автоподстановкой в дневной план.
- История действий и итоги завершённых периодов.
- Настройки уведомлений (вкл/выкл + время).
- Новый UI в темно-золотом стиле с удобной навигацией.

## Запуск

```bash
cd Daily
HOME=/tmp FLUTTER_SUPPRESS_ANALYTICS=true /tmp/appbot_toolchain/flutter/bin/flutter pub get --offline
HOME=/tmp FLUTTER_SUPPRESS_ANALYTICS=true /tmp/appbot_toolchain/flutter/bin/flutter run
```

## Примечание

Для офлайн-сборки в проекте используются только зависимости Flutter SDK.
